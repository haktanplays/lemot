# Grammar Progression

> [!canon] Purpose — Sahiplenilen **fiil/gramer motorlarının (engines)** L1→L15 omurgası ve **hangi sistemlerin bilinçli ertelendiği (deferred)**. Kelime havuzu → [[Vocabulary Progression]]; ticari derinlik → [[Monetization and Scope Boundaries]].

## Executive Summary

Gramer, paradigma tablosuyla değil **split-sense / architecture-verb** kurallarıyla ilerler: her büyük fiil dar bir anlamda aktif olur, gerisi supported/recognition kalır (`lesson-spec-template-v1.1.md:77-83`, `lesson-archetype-templates-v1.md:74-82`). → [[Syllabus Design Rules]].

## Current Canon — engine spine by L15

> [!canon] `L1_L15_CHIP_INVENTORY_AUDIT_2026_07.md:466`: sahiplenilen motor omurgası =
> `je suis · j'ai · je voudrais · je vais · je peux · il faut/je dois · est-ce que · ne ___ pas · y`. **[IMPLEMENTED]**

| Engine | Sahiplenme | Kapsam / kısıt |
|---|---|---|
| `je suis` (être) | L2 | yalnız `je suis` aktif; paradigma recognition; no conjugation table |
| `ne…pas` | L3 | owned fiiller üstünde üretken; negatif kompozisyon `ne ___ pas` frame'i taşır |
| `j'ai` (avoir) | L4 | `j'ai faim/soif/besoin de`; être↔avoir kontrastı |
| `un/une` | L5 | article-light; `le/la` supported; plural/partitive deferred |
| `je vais` (aller) | L7 | movement only; futur proche recognition-only |
| `où` | L8 | tek soru sözcüğü; est-ce que/inversion yok |
| `faire` | L9 | small-action/pause only (split-sense) |
| `je peux` (pouvoir) | L11 | help/permission only (split-sense) |
| `est-ce que` | L12 | owned clause üstüne yes/no wrapper |
| `y` | L14 | place-`y` only; `il y a` ayrı tutulur (homograph guard) |
| `il faut` / `je dois` | L15 | `il faut + inf.` primary (invariable), `je dois + inf.` supported |

## Deferred systems (recognition / not owned)

> [!warning] `band-map:35` (**DEFERRED**): futur proche, full question formation (est-ce que/inversion/çoğu Q-word), `y`/`en` (place-`y` ötesi), broad faire, present `vouloir` (`je veux`), broad pouvoir, `à/au/à la` ötesi konum edatları, tüm geçmiş/gelecek zaman sistemleri, partitives.
> Bunların çoğu **paid-zone owned** — Campfire ~L24'te söz verilir. → [[Monetization and Scope Boundaries]].

## Split-sense pattern (the repeatable engine opening)

Büyük fiil = dar anlam aktif + komşu anlamlar deferred (`lesson-archetype-templates-v1.md:74-82`):
- L7 aller = movement (futur proche deferred)
- L9 faire = small-action (weather/sport/idiom deferred)
- L11 pouvoir = help/permission (broad/conditionnel deferred)
- L15 devoir/falloir = light obligation (`il faut que`+subj deferred)

## Negative-composition debt (guard)

> [!warning] R7 (`audit:298`): `je ne peux pas` (composed), `je n'y vais pas` (deferred), `je ne vais pas` (never shown) hepsi `ne ___ pas` frame'ine biner. Bunlar chip değil, kompozisyon. → [[Chip Taxonomy]].
