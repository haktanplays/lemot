## 2. Owned / Recombined Target

**No new target — the skill is *chaining* owned engines into one human moment**, plus one new meta-phenomenon `phen:can-do-asking` (the "I can ask + I can do" combination).

Recombined engines:
- **`est-ce que + [owned clause]`** (L12) — the standard yes/no question wrapper.
- **`je peux` / `vous pouvez` / `je ne peux pas`** (L11) — help/permission pouvoir, incl. graduated `est-ce que je peux …?` / `est-ce que vous pouvez …?`.
- **`faire une pause` / `faire ça`** (L9) — the small actions you ask to do / can't do.
- **`je vais + place`** (L7) + **`où est … ?`** (L8) — movement + where-control.
- **repair / politeness** (L1): `je ne comprends pas`, `vous pouvez répéter ?`, `c'est pas grave`, `bonjour`, `merci`.

> The lesson's one genuinely new item is the **meta-skill** of combining *can-do* (pouvoir) with *asking* (est-ce que / où) in a single after-class exchange — not a new linguistic system.

## 3. Not Owned / Deferred (explicit)

Recognition-only or deferred — never production targets in L13:
- **`y` / `en`** — `j'y vais` / `on y va` appear **recognition-only** as the **L14 seed hook**; the pronoun is **not owned** (`trap:y-pronoun-too-early`).
- **inversion** — `vas-tu ?`, `allez-vous ?`, `puis-je ?` (recognition).
- **`qu'est-ce que`** + **new question words** (`comment` / `pourquoi` / `quand` / `qui` / `que` / `combien`) — deferred to Question Expansion 2.
- **broad pouvoir** — conditionnel `pourrais`, possibility `il se peut` / `peut-être`, full paradigm (`trap:pouvoir-broad-possibility-too-early`).
- **broad faire** (weather/sport/idioms/`se faire`/full paradigm).
- **object-pronoun system** beyond the frozen `m'aider` chunk.
- **past / future** (passé composé, imparfait, futur proche production).
- **full question system** (reserved post-Campfire ~L24).

## 5. Sentence Family (after-class human scene; known engines only)

> Scene: *"You're after class. You're tired, you want a pause, you need help, and you need to ask whether you can do something or go somewhere."* Statement → est-ce que question transformations are the spine of the lesson.

| Role | Sentence | Note |
|---|---|---|
| **Open** | `Bonjour.` | L1 (recycled) |
| **State** | `Je suis fatigué.` | L2/L6 (recycled) |
| **Need** | `J'ai besoin d'aide.` | L4/L6 (recycled) |
| **Ask for help (statement → question)** | `Vous pouvez m'aider.` → **`Est-ce que vous pouvez m'aider ?`** | L11 chunk wrapped by L12 est-ce que |
| **Ask permission (statement → question)** | `Je peux faire une pause.` → **`Est-ce que je peux faire une pause ?`** | L11 + L12 (graduated) |
| **Can't (decline)** | `Je ne peux pas faire ça.` | L11 (recycled) |
| **Ask about movement (statement → question)** | `Tu vas à la maison.` → **`Est-ce que tu vas à la maison ?`** | L7 wrapped by L12 |
| **Where-question** | `Où est la maison ?` | L8 (recycled) |
| **Movement answer** | `Je vais à la maison.` | L7 (recycled) |
| **Rescue** | `Je ne comprends pas. Vous pouvez répéter ?` | L1/L11 repair |
| **Reassure** | `C'est pas grave.` | L2 (recycled) |
| **Close** | `Merci beaucoup.` | L1 (recycled) |

- **Anchor (chained):** `Je suis fatigué, est-ce que je peux faire une pause ?` (the signature can-do **+** asking line).
- **Interchangeable pieces:** `est-ce que { vous pouvez m'aider / je peux faire une pause / tu vas à la maison } ?` · `je ne peux pas { faire ça }` · `où est { la maison / le café }` · `je vais à { la maison }`.
- **Contrast:** `Vous pouvez m'aider ?` (intonation, L11) vs `Est-ce que vous pouvez m'aider ?` (est-ce que, L12) vs `Pouvez-vous m'aider ?` (**inversion — recognition only**); and `Je vais à la maison.` vs `J'y vais.` (**`y` — recognition only, L14 hook**).
- **Forbidden / not-yet-ready substitutions:** no `y`/`en` production (`j'y vais` recognition only); no inversion (`vas-tu`/`puis-je`); no question words / `qu'est-ce que`; no conditionnel `pourrais` / possibility; no broad faire; no object pronouns beyond `m'aider`; no past/future; no negative wrapped question (`est-ce que je ne …?`).

## 9. Natural Reveal / Feedback

**Chained can-do + asking line (signature):**
- **Expected:** `Je suis fatigué, est-ce que je peux faire une pause ?` / `Est-ce que vous pouvez m'aider ?`
- **Acceptable alternatives:** the intonation forms (`Je peux faire une pause ?`, `Vous pouvez m'aider ?`) stay valid — accept; est-ce que is the neutral standard.

**Common mistakes (watch list):**
- **`est-ce que` on an already-question form** (`Est-ce que tu vas où ?`, `Est-ce que vous pouvez m'aider ?` then adding more) — hint: "`est-ce que` wraps a **statement**, not another question."
- **`je peux aide`** (noun instead of infinitive) → `je peux aider` is in-chunk only; the owned form is `vous pouvez m'aider` — hint: "use the verb inside the help chunk."
- **`je peux fais`** (conjugated after the modal) → `je peux faire` (infinitive) — hint: "after `je peux`, the plain verb."
- **`je peux le faire`** (object pronoun) — object-pronoun system deferred; keep `je peux faire ça`.
- **`y`-pronoun too early** (`j'y vais` as production) — recognition only; "you'll own `y` next lesson; for now `je vais à la maison`."
- **inversion too early** (`puis-je …?`, `vas-tu …?`) — recognition only.
- **overusing `est-ce que` everywhere** (stacking it on every line) — it's the standard wrapper, not mandatory; intonation is still fine.

> Feedback stays passive-mirror; no scores/reward language. Recognition forms (`j'y vais`, inversion, `qu'est-ce que`, `pourrais`) are **redirected, never taught on the spot** (contract §11).

## 12. QA / Pilot Findings

- **Is L13 necessary after L11/L12?** **Yes.** L11 (modal) + L12 (question wrapper) are 2 consecutive control lessons; the rhythm rule calls for a beat before L14 opens `y`. L13 lets the learner *chain* can-do + asking — which neither L11 nor L12 did alone — and proves the engines connect in a real scene.
- **Does it reduce question/pouvoir pressure?** **Yes — its purpose.** 2 active-new (meta, 0 lexis) vs ~26 recycled; load is retrieval/chaining, not acquisition. It consolidates pouvoir + est-ce que before adding the `y` pronoun.
- **Does it prepare L14 `y`-light?** **Yes.** It leaves (a) a movement scene that *wants* a place pronoun, and (b) the recognition hook `j'y vais` beside `je vais à la maison` — the same seed-then-own mechanic L10→L11 used. `trap:y-pronoun-too-early` keeps it recognition.
- **Is novelty too low?** **Borderline — the integration risk, acceptable here.** ≈ 0 new lexis; mitigated by the harder *combination* (can-do + asking) and the L14 `y` forward pull. If smoke shows it feels empty, strengthen the `y` hook, don't add lexis.
- **Does it feel test-like?** **Risk present (archetype #10's named risk), mitigated.** Human after-class scene + Say-It-Your-Way weight + a capability close ("you can ask for what you need"), strictly passive mirror — never a score/quiz.
- **Should L14 remain `y`-light as the band map suggests?** **Yes** — L13 seeds `y` (place sense) cleanly; L14 should own **place `y` only** (`j'y vais` / `on y va`), deferring `en` and other `y` senses (band map §5/§7). L13 does not pre-empt it.
- **Top risks:** (1) **`y` leaking to production** off the hook (guarded by `trap:y-pronoun-too-early` + recognition-only); (2) **est-ce que over-stacking / on already-questions**; (3) **broad-pouvoir creep** (`pourrais`) via the reused `je peux`. All guarded by §3/§9/§10 traps.
