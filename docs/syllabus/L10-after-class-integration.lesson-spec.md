> **Archetype**: primary = **Review / Integration** (#10). Secondary (flavor only, not a second budget) = **Human Context / Small Moment** + **Chunk / Natural Speech** (#1). **New grammar systems: 0. New architecture verbs: 0. Pouvoir ownership: 0.**

## Canon Alignment Note (read first)

Spine: **L0 → L1 → L2 → L3 → L4 → L5 → L6 → L7 → L8 → L9 → L10 (this)**. L6 consolidated L1–L5; then L7 (movement), L8 (where-control), L9 (small-action faire) added three engines in a row. The **pacing guardrail** introduced in the L9 step says: after ~3 consecutive new-engine lessons, insert an **integration / review beat** before opening another major engine. **L10 is that beat** — and it lands exactly where the guardrail predicted.

**What L10 is *not*:**
- **not** a new architecture-verb lesson;
- **not** a `pouvoir` lesson (pouvoir = 0 ownership — only the frozen rescue chunk + one recognition hook; see §2/§3);
- **not** a new tense / aspect / mood doorway;
- **not** a full question-expansion lesson (`où est… ?` / `tu vas où ?` are **recycled**, not re-taught or expanded);
- **not** a new article / preposition lesson;
- **not** a vocabulary-dump (≈ **0 new lexis** — the point is recombination, not acquisition).

**Why a *second* integration lesson is justified (not redundant with L6).** L6 integrated the **foundation** (L1–L5: identity, state, need, articles). L10 integrates a **different, later span** (L6–L9: human context + movement + location-questions + small-action), which only became combinable *after* L9. Two integration beats in ten lessons (L6, L10) establishes a healthy **~every-4–5-lessons rhythm** that the pacing guardrail formalizes. (See §18.)

**Seeding L11 without teaching it.** Just as L6 planted `je voudrais aller` (recognition) to seed L7, **L10 plants the `pouvoir` request shape** — via the recycled frozen `pouvez-vous répéter ?` and one recognition hook `vous pouvez m'aider ?` — to seed **L11 pouvoir-light / help & permission**. `pouvoir` is **met, never produced** here.

**Café-centricity:** café does not appear. The scene is **rest / help after class** (human).

**Legacy mapping divergence (flag).** `CLAUDE.md`'s legacy 24-lesson dev-apk list maps **L10 = Faire**. The **v1 syllabus spine** (this document set, the newer active canon) maps **L9 = Faire-light** and **L10 = Review/Integration (after-class)**. Per the Master Pipeline precedence rule (*newer active canon > codebase canon > older active canon*), this spec follows the **v1 spine**; the legacy bracket is noted, not adopted. Runtime/dev-apk (L1–L5 only) is unaffected — planning canon ahead of runtime.

## 3. Engine Plan

> **No new engine is introduced.** The "primary engine" is **recombination of owned engines** — the integration skill itself (as in L6). Depth labels describe how much *production weight* each cluster carries.

| Engine | Depth | Why it exists |
|---|---|---|
| **A — Mixed known-engine recombination** | **full-cycle** (recombination, not new) | Re-fire L6–L9 engines together in one after-class flow: `je suis + state`, `j'ai besoin de + noun`, `je voudrais + infinitive/noun`, `faire une pause`, `je ne fais pas ça`, `je vais + place`, `où est… ?`, `tu vas où ?`. The lesson's core skill is **chaining** them into a natural moment. |
| **B — Human repair / support scene** | short-cycle | Reuse the rescue + reassurance set naturally inside the exchange: `je ne comprends pas`, `pouvez-vous répéter ?`, `c'est pas grave`, `j'ai besoin d'aide`. |
| **C — L11 `pouvoir` preview (no ownership)** | ambient (recognition / future-hook) | `pouvez-vous répéter ?` (recycled, frozen) sits beside one recognition hook — `vous pouvez m'aider ?` — to *show* that help/permission with `pouvoir` is coming at L11. **Met, not produced.** |

**Why this is an integration beat (not new grammar):** every frame is already owned; the only "new" items are two meta-phenomena; no tense, no verb, no system. The new *skill* is connecting engines across a longer human moment than any single prior lesson required.

**Why this follows L7–L9 naturally:** L7 gave movement, L8 gave where-control, L9 gave small-action — three separate engines. L10 is the first place they **combine** into a believable after-class exchange (tired → need help → want a pause → ask where → say where you're going → recover).

**Why this does not become a `pouvoir` lesson:** `pouvoir` appears only in the recycled frozen rescue chunk and one recognition hook (`vous pouvez m'aider ?`). There is **no** `je peux` / `tu peux` / `on peut`, and **no** productive pouvoir frame. `trap:pouvoir-too-early` guards production.

## 4. Opening Sentence Family

> **Human scene, not a quiz.** *"You're after class. You're tired, you need help, you'd like a pause, you need to ask where something is — and you can say where you're going next."*

| Role | Sentence | Note |
|---|---|---|
| **Open** | `Bonjour.` | L1 greeting (recycled) |
| **State (être)** | `Je suis fatigué.` | L2/L6 state (recycled) |
| **Need (avoir + aide)** | `J'ai besoin d'aide.` | L4/L6 (recycled) |
| **Want a pause (faire)** | `Je voudrais faire une pause.` | L9 (recycled) |
| **Suggest a pause** | `On fait une pause ?` | L9 (recycled, supported) |
| **Refuse (faire)** | `Je ne fais pas ça.` | L9 (recycled) |
| **Where-question (où)** | `Où est la maison ?` | L8 (recycled) |
| **Movement (aller)** | `Je vais à la maison.` | L7 (recycled) |
| **Movement question** | `Tu vas où ?` | L8 (recycled, spoken) |
| **Rescue** | `Je ne comprends pas. Pouvez-vous répéter ?` | L1 rescue (recycled; frozen `pouvoir`) |
| **Reassure** | `C'est pas grave.` | L2 chunk (recycled) |
| **Close** | `Merci beaucoup.` | L1 polite close (recycled) |

- **Interchangeable pieces**: `je suis { fatigué }` · `j'ai besoin de { aide }` · `je voudrais faire { une pause }` · `je ne fais pas { ça }` · `où est { la maison / le café }` · `je vais à { la maison / Paris }` · `tu vas où ?`.
- **Fixed frames (all recycled)**: `frame:j-ai-besoin-de-plus-noun` · `frame:je-voudrais-plus-infinitive` · `chunk:faire-une-pause` · `chunk:je-ne-fais-pas-ca` · `frame:je-vais-plus-place` · `frame:ou-est-plus-noun` · `frame:tu-vas-ou`. *(Recognition only)* `chunk:vous-pouvez-m-aider`.
- **Replaceable slots**: the slots inside each recycled frame (state, need-noun, place, action) — drawn only from the owned closed sets.
- **Anchor**: `Je suis fatigué, je voudrais faire une pause.` (the signature **chained** line) · **Variations**: `J'ai besoin d'aide.`, `Où est la maison ?`, `Je vais à la maison.` · **Contrast**: `Pouvez-vous répéter ?` (owned frozen rescue) vs `Vous pouvez m'aider ?` (**recognition only** — same `pouvoir`, shown side-by-side to seed L11 without owning it).
- **Rescue / natural sentence**: `Je ne comprends pas. Pouvez-vous répéter ?` (+ `c'est pas grave`).
- **Forbidden / not-yet-ready substitutions**:
  - **no `pouvoir` production** (`je peux`, `tu peux`, `on peut`, `vous pouvez m'aider` = recognition only);
  - **no present `vouloir`** (`je veux …`);
  - **no broad faire** (weather/sport/idioms/paradigm);
  - **no full question formation** (`est-ce que`, inversion, other question words);
  - **no new tense / futur proche production / past**;
  - **no new place/location nouns, prepositions, or emotional-vocabulary dump.**

## 10. French-Specific Contrast / Transfer Trap

> Micro-contrasts only (template §7.10) — all **reviews** of prior boundaries.

- **"I need help" → `j'ai besoin d'aide`, not `je suis besoin`.** Need = avoir (L4/L6 boundary, reviewed).
- **"I'd like to take a pause" → `je voudrais faire une pause`.** After `je voudrais`, the verb stays **infinitive** (`faire`) (L6/L9).
- **"I don't do that" → `je ne fais pas ça`.** `ne…pas` wraps `faire` (L9).
- **"Where are you going?" → `tu vas où ?`** Spoken end-placement (L8) — a controlled pattern, **not** the full question system.

## 12. Natural Reveal / Feedback Plan

**Chained after-class line (signature):**
- **Expected**: `Je suis fatigué, je voudrais faire une pause.`
- **Acceptable alternatives**: `Je suis fatigué. J'ai besoin d'une pause.`; split into two sentences; with `s'il vous plaît`.
- **Natural upgrade**: connect with a pause/breath rather than cramming; add `bonjour` to open.
- **Common mistakes**: `je suis besoin` (être for need); `je voudrais fais` (conjugated after voudrais); `je suis vais` (être + aller). **Hint**: "Each engine keeps its own verb: `j'ai besoin`, `je voudrais faire`, `je vais`."
- **Weak-point tags**: `weak:etre-vs-avoir-state`, `weak:voudrais-plus-infinitive`.
- **Passive mirror**: "You put several pieces together — like a real moment."

**Ask for help / where:**
- **Expected**: `J'ai besoin d'aide.` / `Où est la maison ?`
- **Common mistakes**: `Où est-ce que est la maison ?` (full-question-system leak — recognition only). **Hint**: "`Où est … ?` is already a full question."
- **Weak-point**: `weak:question-overload`.

**Pouvoir hook (recognition — feedback is recognition-framed, never a production correction):**
- **Shown**: `Vous pouvez m'aider ?` — "This is how you'll ask for help with `pouvoir`; you'll own it next. For now, `j'ai besoin d'aide` is perfect."
- **If the learner *attempts* `je peux` / `pouvoir`**: accept the meaning, mirror gently, tag `weak:pouvoir-too-early`, defer — **do not** teach `pouvoir` on the spot (contract §11).

**Common mistakes (lesson-wide watch list):** `je suis besoin` · `je voudrais fais une pause` · `je suis vais` · `tu vas où` turning into a full question system · `pouvoir` production too early · overusing café context. **Passive mirror throughout; no scores, no reward language.**

## 18. L10 Pilot Findings

**Does the Review / Integration archetype work here?** **Yes — and it confirms the archetype scales.** L6 proved integration works for the *foundation*; L10 proves it works again for a *later, engine-heavy* span (L6–L9). The learner produces a believable after-class exchange chaining four lessons' worth of engines with **≈ 0 new lexis**. The archetype's value is exactly what the pacing guardrail predicted: consolidation + confidence after an additive run.

**Does L10 confirm the pacing guardrail?** **Yes — directly.** The guardrail (engine §8) was written from the L9 finding ("L7–L9 is a 3-engine run; insert an integration beat"). L10 *is* that beat, and authoring it confirmed the guardrail is actionable: the after-class scene only became possible *because* L7–L9 had each added a piece, and trying to add a fifth engine instead of consolidating would have overloaded the learner. **The guardrail should be treated as a real planning rule, not a footnote** (see patch note below).

**Does it prepare L11 pouvoir-light?** **Yes.** L10 leaves the learner with (a) a help scene that *wants* a permission/ability verb, (b) the recognition hook `vous pouvez m'aider ?`, and (c) owned `faire ça` / `faire une pause` as the actions `pouvoir` will modalize. L11 can open `pouvoir` with almost no new scene-building.

**Should L11 be pouvoir-light / help & permission?** **Yes — recommended.** Scope it split-sense (like L7/L9): own `je peux` / `je ne peux pas` / `on peut … ?` + `vous pouvez m'aider ?`, reusing L9's `faire ça` / `faire une pause`; defer the broad `pouvoir` (probability, polite softening of every request, `il se peut`) and keep `est-ce que je peux … ?` recognition. **After L11, the next new engine should again be followed by an integration beat** (the guardrail rhythm continues).

**Does L10 reveal any required patch to template / archetype / AI contract / ID convention?** Minor, **non-blocking** (propose for review, do **not** auto-apply):
- **Archetype #10 (Review/Integration)** could gain a note from this second pilot: *"Integration lessons recur on a rhythm (≈ every 4–5 lessons, per the pacing guardrail), and may carry a **forward 'preview hook'** for the next engine (L6 → aller hook; L10 → pouvoir hook) to counter the 'too little novelty' risk."*
